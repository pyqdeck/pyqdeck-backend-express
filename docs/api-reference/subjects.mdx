## Overview

Manage the global catalog of academic subjects.

## Endpoints

### List Subjects

`GET /subjects`

### Create Subject

`POST /subjects`

- **Auth**: Required (`admin` or `editor`)

### Update Subject

`PATCH /subjects/:id`

- **Auth**: Required (`admin` or `editor`)

### Delete Subject

`DELETE /subjects/:id`

- **Auth**: Required (`admin`)
