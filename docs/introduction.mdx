## Welcome to the Team!

PyqDeck is a next-generation learning platform designed to help university students excel in their exams. This documentation is for the internal engineering team to understand the system architecture, development standards, and deployment pipelines.

## Our Tech Stack

We use a modern, high-performance stack optimized for developer productivity:

- **Frontend**: Next.js 15, Tailwind CSS 4, shadcn/ui
- **Backend**: Node.js, Express, MongoDB (Prisma)
- **Auth**: Clerk (Custom Branded UI)
- **Design System**: Storybook 8
- **Documentation**: Mintlify
